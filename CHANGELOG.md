# Changelog

Todas as mudanças notáveis deste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [Não Lançado]

### Adicionado

- Sistema de templates de issues para GitHub
- Documentação completa (CONTRIBUTING, SECURITY, CODE_OF_CONDUCT)
- README profissional com estrutura detalhada
- Documentação técnica (ARCHITECTURE.md, API.md, DEPLOYMENT.md)
- Licença MIT

### Alterado

- Reorganização da estrutura de documentação

## [1.0.0] - 2026-01-XX

### Adicionado

- Curso completo com 10 módulos
- 48 aulas em formato MDX
- Playground interativo para código C
- Sistema de navegação progressiva
- Suporte a vídeos do YouTube incorporados
- API backend com Elysia
- Rate limiting e validação de inputs

### Tecnologias

- Frontend: Astro 5.x, MDX, TypeScript
- Backend: Bun, Elysia
- Editor: Monaco
